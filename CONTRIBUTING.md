# Contributing

Quick guidelines:
- Target framework: .NET Framework 4.7.2, C# 7.3.
- Indentation: 4 spaces. Follow `.editorconfig` if added.
- Include all designer and resource files when committing.
- Mark runtime resources (e.g. `Resources/sample.csv`) as Content and CopyToOutputDirectory=Always.

Workflow:
- Create a branch per feature/fix.
- Commit with clear messages and open a PR for review.
